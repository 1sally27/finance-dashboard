구글 시트 연동 설정

데이터를 구글 스프레드시트에 저장하면 어떤 기기에서도 최신 데이터를 불러올 수 있어요.

  1. 구글 스프레드시트를 새로 만드세요
  2. 상단 메뉴 → 확장 프로그램 → Apps Script
  3. 제공된 Code.gs 내용을 붙여넣기
  4. 저장 후 initSheets 함수 실행 (시트 초기화)
  5. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자
  6. 생성된 웹 앱 URL을 아래에 붙여넣기
